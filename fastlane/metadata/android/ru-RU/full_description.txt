<i>Materialious</i> — это клиент для <a href='https://github.com/iv-org'>Invidious</a>, использующий Material Design.
Позволяет просматривать и смотреть видео, управлять плейлистами и использовать каналы.

<b>Возможности:</b>
<ul>  
  <li>Поддержка Invidious Companion</li>  
  <li>Расширенная интеграция с API Invidious</li>  
  <li>Резервное использование YouTube.js, если Invidious не загружает видео (для ПК и Android)</li>  
  <li>Возможность отключения проверки сертификатов для пользователей домашних серверов</li>  
  <li>Синхронизация прогресса просмотра между сессиями Invidious</li>  
  <li>Совместный просмотр (watch sync parties)</li>  
  <li>Встроенный SponsorBlock</li>  
  <li>Встроенный Return YouTube Dislikes</li>  
  <li>Встроенный DeArrow (с локальной обработкой при сбоях подключения)</li>  
  <li>Без рекламы</li>  
  <li>Без трекинга</li>  
  <li>Светлая и тёмная темы</li>  
  <li>Пользовательские цветовые темы</li>  
  <li>Интеграция с подписками, историей просмотров Invidious и другими функциями</li>  
  <li>Поддержка прямых трансляций</li>  
  <li>Поддержка DASH</li>  
  <li>Поддержка глав в видео</li>  
  <li>Режим «Только аудио»</li>  
  <li>Поддержка плейлистов</li>  
</ul>
