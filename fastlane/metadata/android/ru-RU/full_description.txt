<i>Materialious</i> — это клиент для <a href='https://github.com/iv-org'>Invidious</a>, использующий Материальный дизайн.
Он позволяет просматривать и смотреть видео, управлять плейлистами и использовать каналы.

<b>Возможности:</b>
<ul>
  <li>Поддержка компаньона Invidious!</li>
  <li>Интеграция с расширенным API Invidious!</li>
  <li>Резервное использование YouTube.js если Invidious не может загрузить видео в настольной версии или на Адроиде</li>
  <li>Можно отключить проверку сертификатов TLS для серверов развёрнутых дома</li>
  <li>Синхронизация вашего прогресса просмотра между сеансами Invidious</li>
  <li>Групповые просмотры!</li>
  <li>Встроенный SponsorBlock для пропуска интегрированной рекламы.</li>
  <li>Возврат дизлайков на YouTube.</li>
  <li>Встроенный DeArrow (с поддержкой локальной обработки видео).</li>
  <li>Без рекламы.</li>
  <li>Без следилок.</li>
  <li>Светлые и тёмные темы оформления.</li>
  <li>Пользовательские цветовые темы.</li>
  <li>Интеграция с подписками Invidious, историей просмотра и больше.</li>
  <li>Поддержка прямых трансляций.</li>
  <li>Поддержка Dash.</li>
  <li>Главы.</li>
  <li>Режим «Только звук».</li>
  <li>Плейлисты.</li>
</ul>
